# Mini-E-commerce

Aplicación web que simula una tienda online desarrollada en JavaScript puro, enfocada en la gestión del carrito, renderizado dinámico de productos y persistencia de datos.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
  
## Funcionalidades

- Visualización dinámica de productos
- Añadir y eliminar productos del carrito
- Cálculo automático del total
- Persistencia del carrito mediante LocalStorage
- Diseño Responsive

## Objetivos del proyecto

Reforzar fundamentos de JavaScript trabajando manipulación del DOM, eventos, estado y arquitectura modular sin utilizar frameworks.
